🐿️ NYC Squirrel Census Data Analyzer
A Python data analysis project that processes real New York City
squirrel census data and counts squirrels by fur color using Pandas.
📋 About
This project analyzes the 2018 Central Park Squirrel Census dataset
from NYC Open Data. It counts squirrels by fur color (Gray, Cinnamon,
and Black) and exports the results to a clean CSV file.
🖥️ Example Output
```
Grey squirrels: 2473
Red squirrels: 392
Black squirrels: 103
CSV saved!
```
📊 Output CSV
```
Fur Color,Count
Gray,2473
Cinnamon,392
Black,103
```

🚀 How to Run
Make sure Python is installed
Install Pandas:
```
pip install pandas
```
Clone this repository:
```
git clone https://github.com/alexisn1989/squirrel-census-analyzer
```
Run the program:
```
python main.py
```
